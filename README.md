
# Voting Eligibility Checker

A simple web application built using HTML, CSS, and JavaScript that checks whether a user is eligible to vote based on their age.

## Features

* User-friendly interface
* Age validation
* Instant eligibility result
* Responsive design
* Hover effects and box shadow styling

## Technologies Used

* HTML5
* CSS3
* JavaScript


## How It Works

1. Enter your age in the input field.
2. Click the **Click** button.
3. The application checks the entered age.
4. It displays:

   * "You are Eligible to Vote" if age is 18 or above.
   * "You are not Eligible to Vote" if age is below 18.
   * "Please enter your age" if no age is entered.

## Future Improvements
